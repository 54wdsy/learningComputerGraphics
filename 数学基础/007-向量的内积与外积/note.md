### 一、向量点乘

可以这样写:
<img src="01.gif">
```js
a · b = |a| × |b| × cos(θ)
```

也可以这样:
<img src="02.gif">
```js
a · b = ax * bx + ay * by
```

两种方法是相同的。
<img src="03.gif">




### 二、向量叉乘
可以这样计算:
<img src="05.gif">
```js
a × b = |a| |b| sin(θ) n
```

也可以这样计算:
<img src="06.8">

```js
cx = aybz − azby
cy = azbx − axbz
cz = axby − aybx
```
